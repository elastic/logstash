### Changes to SSL settings in {{ls}} plugins [ssl-settings-9.0]
Weâ€™ve removed deprecated SSL settings in some {{ls}} plugins, and have replaced them with updated settings. If your plugin configuration contains any of these obsolete options, the plugin may fail to start.

### Enterprise_search integration plugin is deprecated [enterprise_search-deprecated-9.0]

Weâ€™ve deprecated the {{ls}} Enterprise_search integration plugin, and its component App Search and Workplace Search plugins. These plugins will receive only security updates and critical fixes moving forward.

We recommend using our native {{es}} tools for your Search use cases. For more details, please visit [Search UI with Elasticsearch](search-ui://tutorials-elasticsearch.md).
